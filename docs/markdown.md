1、标题

    # 一级标题
    ## 二级标题
    ### 三级标题
    以此类推，总共六级标题

2、列表

    有序列表：文字前面加上 * 或者 -
    * 1
    * 2
    * 3

    无序列表：列表前面加上1. 2. 3. 数字与内容之间要有一个空格
    1. a
    2. b
    3. c

3、引用

    引用小段别处句子需要用引用格式，只需在句子前加上 >
    > 例如这样

4、图片和链接

    插入链接：[baidu](http://www.baidu.com/"title")
    插入图片：![图片名](图片链接)

5、粗体和斜体
    
    粗体：**这里面是粗体**
    斜体：*这里面是斜体*
